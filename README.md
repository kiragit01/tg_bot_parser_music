# 🎵 ym-telegram-bot

Telegram бот для переноса треков из Яндекс.Музыки — **без цензуры**.

Берёт список треков из плейлиста ЯМ (или текстовый список), ищет на YouTube через yt-dlp,
скачивает аудио и отправляет в Telegram. Поддерживает 2000+ треков с параллельной обработкой.

## Быстрый старт

```bash
# 1. Создай .env из примера
cp .env.example .env
# Впиши токен от @BotFather в TELOXIDE_TOKEN

# 2. Запусти
cargo run

# С подробным логированием:
RUST_LOG=debug cargo run
```

При первом запуске автоматически скачаются бинарники yt-dlp и ffmpeg в папку `libs/`.

## Использование

В Telegram:
- `/start` — приветствие
- `/help` — список команд
- `/parse <ссылка>` — парсить плейлист Яндекс.Музыки
- `/get Исполнитель - Название` — скачать один трек
- `/downloadall` — скачать все треки из последнего распарсенного плейлиста
- Или просто отправь ссылку на плейлист / текстовый список треков

### Текстовый список

Можно отправить боту список в формате:
```
Кишлак - Дурак
Скриптонит - Животные
Пошлая Молли - Нет Монет
```

Бот распознает треки и начнёт параллельное скачивание.

## Структура проекта

```
src/
├── main.rs          — точка входа, инициализация бота и yt-dlp
├── commands.rs      — команды бота (enum + BotCommands)
├── handlers.rs      — обработчики сообщений, параллельное скачивание
├── models.rs        — структуры данных (Track, Playlist)
├── yandex.rs        — парсинг плейлистов ЯМ через JSON API
└── downloader.rs    — поиск и скачивание треков через yt-dlp
```
