# 🎵 YM Telegram Bot

Telegram-бот для скачивания музыки из **любых источников** прямо в чат — быстро, параллельно, без ограничений.

Кинул ссылку на плейлист Яндекс.Музыки или просто текстовый список треков — бот сам найдёт и отправит аудио. Работает с VK, SoundCloud, YouTube и Яндекс.Музыкой одновременно.

---

## Что умеет

**Импорт плейлистов**
- Ссылки на плейлисты Яндекс.Музыки — автоматический парсинг
- Iframe-код (кнопка «Код для вставки» в ЯМ)
- Текстовый список треков в формате `Исполнитель - Название`

**Поиск и скачивание**
- Мультиплатформенный поиск: VK → SoundCloud → YouTube → Яндекс.Музыка
- Умная приоритизация: сначала VK/SC (лучше качество, быстрее), потом YouTube
- Параллельная загрузка до 10 треков одновременно (настраивается)
- Поддержка плейлистов до 2000+ треков
- Выборочное скачивание: `/download 30-60` или `/download 1,5,10`

**Тексты песен**
- Интеграция с Genius — тексты на языке оригинала
- Кнопка «Текст» под каждым скачанным треком
- Команда `/lyrics Исполнитель - Название` для отдельного поиска
- Отображение в свёрнутых цитатах (не засоряет чат)

**Кэширование**
- Если трек уже был отправлен — мгновенная пересылка без повторного скачивания
- Кэш поисковых запросов (30 дней) — повторные поиски работают мгновенно

**Форумы и топики**
- В супергруппах с топиками бот отвечает в той же теме, где получил команду
- Опция создания нового топика для каждого плейлиста

**Настройки**
- `/settings` — включение/выключение текстов песен
- Настройки сохраняются для каждого пользователя

## Команды

| Команда | Описание |
|---|---|
| `/start` | Приветствие и краткая справка |
| `/help` | Список всех команд |
| `/get Исполнитель - Название` | Найти и скачать один трек |
| `/parse <ссылка>` | Распарсить плейлист Яндекс.Музыки |
| `/downloadall` | Скачать все треки из последнего плейлиста |
| `/download 30-60` | Скачать треки по номерам или диапазону |
| `/stop` | Остановить текущее скачивание |
| `/status` | Показать статус активной задачи |
| `/lyrics Исполнитель - Название` | Найти текст песни |
| `/settings` | Настройки бота |

Также можно просто отправить боту:
- Ссылку на плейлист ЯМ — бот распарсит автоматически
- Текстовый список треков — бот начнёт скачивание сразу
- `Исполнитель - Название` — скачает один трек

## Быстрый старт

### 1. Клонируй и настрой

```bash
git clone <repo-url>
cd ym-telegram-bot

cp .env.example .env
# Заполни .env — обязателен только TELOXIDE_TOKEN
```

### 2. Запусти

```bash
cargo run
```

При первом запуске автоматически скачаются `yt-dlp` и `ffmpeg` в папку `libs/`.

### 3. Готово

Открой бота в Telegram и отправь ссылку на плейлист или трек.

## Настройка (.env)

| Переменная | Обязательно | Описание |
|---|---|---|
| `TELOXIDE_TOKEN` | да | Токен бота от [@BotFather](https://t.me/BotFather) |
| `VK_TOKEN` | нет | Токен VK (лучшее качество, без цензуры). Можно несколько через запятую |
| `SC_OAUTH_TOKEN` | нет | OAuth-токен SoundCloud. Можно несколько через запятую |
| `GENIUS_TOKEN` | нет | Токен Genius API для текстов песен. Можно несколько через запятую |
| `YM_TOKEN` | нет | Токен Яндекс.Музыки (прямое скачивание, но может быть цензура) |
| `MAX_CONCURRENT` | нет | Параллельность скачивания (по умолчанию 10) |
| `RUST_LOG` | нет | Уровень логов: `info` (по умолчанию), `debug`, `trace` |

> Без `VK_TOKEN` и `SC_OAUTH_TOKEN` бот будет искать только на YouTube — это работает, но медленнее.

### Как получить токены

<details>
<summary><b>VK_TOKEN</b> — токен VK Music</summary>

1. Перейди по ссылке (авторизация через Kate Mobile):
   ```
   https://oauth.vk.com/authorize?client_id=2685278&scope=audio&response_type=token&v=5.131
   ```
2. Залогинься и разреши доступ
3. Из URL после редиректа скопируй `access_token=ТВОЙ_ТОКЕН`

> Токен привязан к IP. Получай с того же IP, где запущен бот.
</details>

<details>
<summary><b>SC_OAUTH_TOKEN</b> — токен SoundCloud</summary>

1. Открой [soundcloud.com](https://soundcloud.com) и залогинься
2. DevTools (F12) → Network → фильтр `api-v2`
3. Кликни на любой запрос к `api-v2.soundcloud.com`
4. В заголовках найди `Authorization: OAuth XXXXX`
5. Скопируй `XXXXX` (без `OAuth `)
</details>

<details>
<summary><b>GENIUS_TOKEN</b> — токен Genius (тексты песен)</summary>

1. Зайди на [genius.com/api-clients/new](https://genius.com/api-clients/new)
2. App Name — любое, App Website URL и Redirect URI — `http://localhost`
3. Save → скопируй **Client Access Token**
</details>

<details>
<summary><b>YM_TOKEN</b> — токен Яндекс.Музыки</summary>

Токен Яндекс.Музыки для прямого скачивания. Без него бот парсит плейлисты ЯМ, но скачивает с других платформ.
</details>

## Как это работает

```
Пользователь                     Бот
    │                              │
    ├─ Ссылка на плейлист ЯМ ─────►│
    │                              ├─ Парсит JSON API Яндекс.Музыки
    │                              ├─ Показывает список треков
    │                              ├─ Ищет каждый трек: SC → VK → YT
    │◄─ Аудио-файлы ──────────────┤   (параллельно, до 10 штук)
    │                              │
    ├─ /get Кишлак - Дурак ───────►│
    │                              ├─ Ищет на всех платформах
    │                              ├─ Показывает найденные варианты
    │◄─ Выбранный трек ───────────┤
    │                              │
    ├─ /lyrics Oxxxymiron - Где.. ►│
    │                              ├─ Ищет на Genius
    │◄─ Текст песни ──────────────┤
```

### Приоритет источников (режим «Авто»)

1. **SoundCloud** — быстро, хорошее качество
2. **VK Music** — без цензуры, оригинальные версии
3. **YouTube** — самая большая база
4. **Яндекс.Музыка** — fallback (может содержать цензурированные версии)

## Структура проекта

```
src/
├── main.rs          — точка входа, инициализация
├── commands.rs      — определение команд бота
├── handlers.rs      — обработчики сообщений и callback-кнопок
├── models.rs        — структуры данных (Track, Playlist, SearchResult)
├── yandex.rs        — парсинг плейлистов Яндекс.Музыки
├── downloader.rs    — мультиплатформенный поиск и скачивание
├── cache.rs         — SQLite-кэш (треки, настройки, поиск)
├── lyrics.rs        — получение текстов с Genius
├── tokens.rs        — пул токенов с ротацией
├── vk.rs            — скачивание из VK Music
└── ym.rs            — скачивание из Яндекс.Музыки
```

## Технологии

- **Rust** + [teloxide](https://github.com/teloxide/teloxide) — асинхронный Telegram Bot API
- **yt-dlp** — поиск и скачивание с YouTube/SoundCloud
- **SQLite** (rusqlite) — кэширование треков и настроек
- **Genius API** — тексты песен с фильтрацией переводов
- **VK API** — доступ к аудио VK Music
- **Яндекс.Музыка API** — парсинг плейлистов и прямое скачивание
